Features of HTML

1. Each HTML tag is enclosed within a < (left bracket) and > (right bracket). These special characters differentiate tags from ordinary text.
2. Opening tags are denoted by, for example, <I>. This is the opening tag to display text in italics.
3. Closing tags are denoted by, for example, </I>. This is the closing italics tag.
4. Tag names are not case sensitive, but as a convention, they are written in upper case to distinguish them from normal text.
